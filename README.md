# Fitness Tracker

Fitness Tracker is an Android app I worked on as part of a school group project for my Intro to Software Engineering course. The goal of the project was to build a simple fitness tracker where users can create workouts, add exercises, track sets and reps, review previous workouts, and recieve advice from an A.I. assistant.

This project was also my first real experience working on a larger Android codebase with multiple people, so a big part of the project was learning how to organize features, connect different screens together, use Firebase, and manage changes through GitHub.

## Features

- User sign up and login with Firebase Authentication
- Workout creation flow
- Add exercises to a workout
- Track sets, reps, and weight for exercises
- Save completed workouts
- View previously completed workouts
- Basic profile/view screens
- Custom bottom navigation between main app pages
- AI chat screen structure for a fitness assistant feature

## Tech Stack

- **Kotlin** — main programming language
- **XML** — Android UI layouts
- **Firebase Authentication** — user login and account creation
- **Firebase Firestore** — storing user workout data
- **Android Studio** — development environment
- **Git/GitHub** — version control and team collaboration

## Setup Notes

This public version of the project does not include private Firebase or API credentials.

To run the project locally, you would need to:

1. Clone the repository.
2. Open the project in Android Studio.
3. Create your own Firebase project.
4. Add your own `google-services.json` file inside the `app/` folder.
5. Add any required API keys or configuration values.
6. Build and run the app on an emulator or Android device.
